# Cédulas

Esta carpeta contiene las cédulas de trabajo, análisis y documentación de soporte del proyecto.

## Tipos de archivos

- **Cédulas de análisis**: Documentos de análisis detallado por área o proceso (`.xlsx`, `.csv`, `.md`)
- **Cédulas de validación**: Registros de verificación y validación de datos
- **Cédulas de seguimiento**: Documentos de seguimiento de tareas, hallazgos y pendientes
- **Cédulas de auditoría**: Archivos de respaldo para revisiones y auditorías

## Convenciones de nomenclatura

Los archivos deben seguir el formato: `cedula_[area]_[descripcion].[extensión]`

Ejemplo: `cedula_finanzas_conciliacion-enero.xlsx`
