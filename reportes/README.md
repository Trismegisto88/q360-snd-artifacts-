# Reportes

Esta carpeta contiene los reportes generados por el proyecto, tanto automáticos como manuales.

## Tipos de archivos

- **Reportes periódicos**: Informes diarios, semanales, mensuales y anuales (`.pdf`, `.xlsx`, `.html`)
- **Reportes de análisis**: Documentos con análisis profundo de datos y resultados
- **Reportes de ejecución**: Logs y resultados de procesos automáticos
- **Reportes de excepciones**: Documentos que registran errores, alertas o casos atípicos
- **Plantillas de reportes**: Templates reutilizables para la generación de informes

## Convenciones de nomenclatura

Los archivos deben seguir el formato: `reporte_[tipo]_[periodo].[extensión]`

Ejemplo: `reporte_ventas_2026-01.pdf`
