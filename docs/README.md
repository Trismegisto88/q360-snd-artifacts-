# Docs (Documentación)

Esta carpeta contiene la documentación general del proyecto.

## Tipos de archivos

- **Manuales de usuario**: Guías e instrucciones para el uso del sistema (`.md`, `.pdf`)
- **Documentación técnica**: Guías de instalación, configuración y despliegue
- **Políticas y procedimientos**: Documentos normativos y procedimentales del proyecto
- **Glosarios**: Definición de términos, acrónimos y conceptos clave
- **Actas y minutas**: Registros de reuniones y acuerdos del equipo
- **Historial de cambios**: Registro de versiones y modificaciones del proyecto (`CHANGELOG.md`)

## Convenciones de nomenclatura

Los archivos deben seguir el formato: `[tipo]_[descripcion].[extensión]`

Ejemplo: `manual_instalacion-local.md`
