# Specs (Especificaciones)

Esta carpeta contiene las especificaciones técnicas y funcionales del proyecto.

## Tipos de archivos

- **Especificaciones funcionales**: Descripción detallada de los requerimientos y funcionalidades (`.md`, `.pdf`, `.docx`)
- **Especificaciones técnicas**: Documentación técnica de arquitectura, APIs e integraciones
- **Casos de uso**: Definición de flujos y escenarios de uso del sistema
- **Criterios de aceptación**: Condiciones que deben cumplirse para validar cada funcionalidad
- **Diagramas**: Diagramas de flujo, secuencia y arquitectura (`.drawio`, `.png`, `.svg`)

## Convenciones de nomenclatura

Los archivos deben seguir el formato: `spec_[modulo]_[descripcion].[extensión]`

Ejemplo: `spec_autenticacion_flujo-login.md`
