# Dashboards

Esta carpeta contiene los archivos de paneles de control e informes visuales del proyecto.

## Tipos de archivos

- **Paneles de control**: Archivos de configuración y definición de dashboards (`.json`, `.yaml`, `.pbix`, `.twb`)
- **Visualizaciones**: Gráficos, tablas y componentes visuales reutilizables
- **Configuraciones de métricas**: Definición de KPIs y métricas clave del negocio
- **Layouts**: Plantillas y disposiciones de elementos en los paneles

## Convenciones de nomenclatura

Los archivos deben seguir el formato: `dashboard_[nombre-descriptivo].[extensión]`

Ejemplo: `dashboard_ventas-mensuales.json`
